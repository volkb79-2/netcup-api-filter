API Quick Reference
Authentication

Include your API token in the Authorization header:

Authorization: Bearer naf_username_xxxxxxxxxxxx...
Endpoints
GET /api/dns/records List DNS records for your realm
PUT /api/dns/records Update DNS record value
POST /api/dns/records Create new DNS record
DELETE /api/dns/records Delete DNS record
Example: Update A Record
curl -X PUT https://api.example.com/api/dns/records \
  -H "Authorization: Bearer naf_user_xxx..." \
  -H "Content-Type: application/json" \
  -d '{"hostname": "home.example.com", "type": "A", "value": "1.2.3.4"}'
